# cpp-cli-blade-multi-file-hello-world

## Description
A POC for blade build system demostrates header files
for c/cpp.

## Tech stack
- cpp
- g++
- ninja
- blade

## Docker stack
- ubuntu:22.04

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
